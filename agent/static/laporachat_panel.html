Demo en vivo (10 min) — el guion

1. El dolor — "¿Cuántos DMs se te quedan sin responder? ¿Cuántas citas no llegan?"
2. El comentario mágico — comenta "GUIA" en vivo → respuesta pública + DM personalizado en segundos
3. La IA agenda — "¿tienes el lunes a las 3?" → consulta el calendario real → cita con anticipo de $20.000 al Nequi
4. El CRM — el contacto aparece con su conversación completa
5. Cierre — "Esto, con tu marca y tu Nequi, te lo entrego andando en una semana: $350k de montaje + $450k/mes. ¿Arrancamos con tu próximo video?"

Instalación por cliente (resumen del playbook — 2-3 horas)

Meta: invitar su cuenta como Evaluador de Instagram → él acepta → conectar cuenta → generar token → activar suscripción webhook
n8n: credencial Query Auth con su token · automatizaciones con campo cliente · su Google Calendar · su horario/Nequi en el prompt
CRM: clínica del cliente + API key (lectura+escritura) + credencial Header Auth en n8n
Pruebas: comentario → respuesta+DM · anti-duplicados · cita en calendario+CRM · follow gate
⚠ Trampas conocidas y detalles completos: PLAYBOOK-INSTALACION.md en plantilla-clientes/. Sin App Review de Meta, solo cuentas con rol de evaluador disparan eventos (máx. 50 — suficiente para los primeros clientes).

Conexiones

Estado real de cada canal. Sin humo: lo que funciona, lo que es parcial y lo que la plataforma no permite.

Instagram

Falta token

Motor listo y publicado en n8n. Comentario con palabra clave → respuesta pública rotativa + DM con link + lead. DMs entrantes los responde Gemini. Solo falta conectar tu app de Meta (pestaña Instalación).

Facebook

Falta token

Motor listo. Usa la misma app de Meta que Instagram: comentarios en publicaciones de tu página → respuesta + mensaje privado por Messenger.

WhatsApp

Disponible

Con la Cloud API (la del bot de zapatillas) el link del DM puede llevar directo a un chat de WhatsApp con respuesta IA — el canal que más convierte en Colombia.

TikTok

API cerrada

TikTok no ofrece API pública para responder comentarios ni DMs (ManyChat la tiene por alianza directa). La jugada que sí funciona: el CTA del video de TikTok manda a comentar la palabra clave en tu Instagram — CTA Studio te genera ese guion.

YouTube

Parcial

La API de YouTube permite auto-responder comentarios con tu link (integrable con tu ecosistema Google), pero no existen DMs. Se entrega el link en la respuesta pública.

Gemini (Google)

Conectado

Tu credencial de Google Gemini ya está enlazada al motor. Responde DMs con memoria por usuario y saluda por el nombre con la variable {nombre}.
